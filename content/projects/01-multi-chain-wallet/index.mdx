# 项目1：多链钱包 + 账户抽象(ERC-4337)

## 项目概述

这是Web3 Dev Lab的第一个项目，专注于**密钥管理**、**交易签名**和**账户抽象(ERC-4337)**的完整实现。

### 目标

完成本项目后，你将能够：

✅ 理解密钥对的生成、存储和加密
✅ 掌握ECDSA签名和EIP-712签名机制
✅ 实现完整的钱包逻辑（生成、导入、交易）
✅ 理解UserOperation和Bundler的工作原理
✅ 实现白名单Paymaster进行Gas赞助
✅ 构建支持多链的钱包系统

### 核心要素

| 主题 | 内容 |
|------|------|
| 🔑 **密钥管理** | 生成、存储、加密、导入 |
| ✍️ **签名机制** | ECDSA、EIP-191、EIP-712 |
| 📡 **RPC交互** | 多链RPC管理、交易发送 |
| 🔐 **ERC-4337** | UserOp、Bundler、Paymaster |
| 🌍 **多链支持** | Ethereum、Polygon、Arbitrum等 |

### 项目时长

⏱️ **3周** (可根据你的进度调整)

### 技术栈

- **后端**：Go + Geth库
- **合约**：Solidity (SimpleAccount + Paymaster)
- **前端**：Next.js + Wagmi (可选)
- **测试**：Foundry + Go测试
- **部署**：Docker + Makefile

### 成果物

完成此项目，你将拥有：

- 📝 完整的架构文档和设计图
- 🔧 可运行的Go钱包实现
- 📋 Solidity合约(SimpleAccount、Paymaster)
- 🧪 完善的测试套件(单测+集成测试)
- 📊 本地Prometheus+Grafana监控
- ✅ Docker一键启动环境
- 📹 30-60秒演示视频
