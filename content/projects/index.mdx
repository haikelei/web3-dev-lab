## 推荐学习顺序

### 🟢 最小可行路线 (3个项目 = 9周)
```
项目1 → 项目2 → 项目3
钱包   →  DEX  → 借贷
```
**适合**：想快速掌握Web3核心的开发者

### 🟡 完整进阶路线 (5个项目 = 12周)
```
项目1 → 项目2 → 项目3 → 项目4 → 项目5
钱包   →  DEX  → 借贷  → 跨链  →  ZK
```
**适合**：想成为Web3高级工程师的开发者

### 🔴 专项深化 (选一个深学)
- 完成项目1/2/3后，选择项目4或5其中之一深度研究
- 补充索引器/MEV模拟等进阶话题
